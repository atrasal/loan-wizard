Problem Statement 3 — Agentic AI Video Call-Based Onboarding

Video-Based Digital Loan Origination & Risk Assessment

An AI-powered, end-to-end loan onboarding system that uses live video calls for real-time KYC, risk assessment, and instant personalized offer generation — all within a single browser session.

🏆 Team AiMatrix   |   Hackathon Submission   |   April 2026

01 — THE PROBLEM

Why Traditional Loan Journeys Fail

  • 60-70% drop-off rates in purely form-based digital applications
  • Fraud & misrepresentation risks with no real-time identity verification
  • Manual KYC overhead — physical document collection adds days to processing
  • No contextual understanding — forms capture data but miss customer intent
  • Compliance gaps — incomplete audit trails and consent records

03 — ARCHITECTURE

Technology Stack

Presentation Layer — Vite + Vanilla JS SPA, Glassmorphism UI, Outfit + Inter fonts
AI / ML Layer — Web Speech API (STT), @vladmandic/face-api (Age/Gender), Browser Geolocation
Processing Layer — Risk Engine (10 rules), LLM Intelligence (classification), Offer Engine (EMI calc)
Data & Audit Layer — MediaRecorder (video), Central Logger, JSON Export, Consent Trail
Vite 6 ES Modules Web Speech API face-api.js MediaRecorder Geolocation API Lucide Icons TensorFlow.js

Why Browser-First?

  • Zero latency — No round-trips to server
  • Data privacy — Video never leaves device
  • Instant deployment — Just a URL, no app install
  • Infinite scale — Each user is their own compute node
  • Works offline — Core features after initial load
04 — CORE FEATURE

AI-Powered Video KYC

🎤 Real-Time Speech-to-Text

  • Web Speech API with continuous recognition (en-IN)
  • NLP entity extraction: name, income, employment, loan purpose
  • Automatic consent phrase detection
  • Live transcript with confidence scores

👤 Computer Vision — Face Analysis

  • TinyFaceDetector for real-time detection (500ms intervals)
  • Age estimation with 15-frame moving average smoothing
  • Gender classification with probability score
  • Liveness tracking (face-present frame percentage)

📍 Geolocation Capture

  • High-accuracy GPS coordinates via browser API
  • Reverse geocoding to city/state (BigDataCloud)
  • Location serviceability verification
  • Fraud signal: geo-mismatch detection

🎬 Session Recording & Audit

  • MediaRecorder API captures video + audio
  • WebM format with VP9 codec (1.5 Mbps)
  • Playback in audit dashboard
  • Downloadable for regulatory compliance

09 — EVALUATION

Judging Criteria Alignment

✓ End-to-End Digitisation

7-step fully paperless journey. Customer goes from campaign link → approved offer in a single 3-5 minute video session. Zero manual intervention required. No physical documents needed.

✓ Accuracy & Compliance

Adheres to RBI e-KYC norms. Complete audit trail with video recording, timestamped transcripts, verbal consent capture, geolocation evidence, and exportable JSON logs for regulatory review.

✓ Risk Mitigation

5 fraud signal checks (location verification, age consistency, liveness detection, consent verification, session integrity) + 5 policy rules + simulated CIBIL bureau scoring. Age mismatch & geo-anomaly detection built-in.

✓ Intelligence & Personalization

LLM classifies customer persona, risk band, lifetime value, and intent. Offers are personalized across 3 tiers with income-based multipliers, bureau-adjusted interest rates, and interactive EMI configuration.

✓ Scalability & Reliability

All AI processing runs in the browser — each user is their own compute node. Zero server load for video/STT/face detection processing. Architecture supports infinite concurrent users. Sub-second latency for all real-time decisions. Static deploy (Vercel/Netlify) with global CDN.
